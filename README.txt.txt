If you have Node.js and NPM installed you can run the Discrimination Barometer locally on your machine.
(1) Move to the directory '/discrimination-barometer'.
(2) In the terminal run 'npm run serve'.
(3) The app will be running on http://localhost:8080/ or whatever port is available on your machine. 

If you do not have Node.js and NPM installed on your machine you can access an online version here: 
https://diskriminierungs-barometer.netlify.app/
